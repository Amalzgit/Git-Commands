Git Cheat Sheet

Basic Commands

  • git init
    Initialize a new repository
  • git clone <repository-url>
    Copy an existing remote repository
  • git status
    Show the status of the working directory and staging area
  • git add <file>
    Stage changes for the next commit (Use 
    git add .
    to stage all)
  • git commit -m "message"
    Record staged changes with a message
  • git push
    Push local commits to the remote repository
  • git pull
    Fetch and merge changes from the remote repository
  • git fetch
    Download objects and refs from another repository

Branching and Merging

  • git branch
    List local branches
  • git branch <branch-name>
    Create a new branch
  • git checkout <branch-name>
    Switch to a branch
  • git merge <branch-name>
    Merge changes from a branch into the current branch
  • git branch -d <branch-name>
    Delete a branch
  • git branch -M <new-branch-name>
    Rename the current branch

Viewing History

  • git log
    Show commit history
  • git diff
    Show changes between working directory and staging area
  • git show <commit>
    Display information about a specific commit

Undoing Changes

  • git reset <file>
    Unstage a file, keeping changes
  • git checkout -- <file>
    Discard changes in the working directory
  • git revert <commit>
    Undo changes from a specific commit

Configuration

  • git config --global user.name "Your Name"
    Set user name
  • git config --global user.email "your.email@example.com"
    Set user email

Stashing Changes

  • git stash
    Temporarily save changes
  • git stash pop
    Restore and remove the most recent stash

Tagging

  • git tag <tag-name>
    Create a tag at the current commit

Remote Repository Management

  • git remote -v
    List remote repositories
  • git remote add <name> <url>
    Add a remote repository
  • git remote remove <name>
    Remove a remote repository

Advanced Commands

  • git cherry-pick <commit>
    Apply changes from a specific commit
  • git rebase <branch>
    Reapply commits onto another branch
  • git reset --hard <commit>
    Discard all changes to match a specific commit
  • git reflog
    Show a log of actions in the repository
  • git bisect
    Find the commit that introduced a bug
